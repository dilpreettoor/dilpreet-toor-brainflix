## Run Locally

Clone the project

```bash
  git clone https://github.com/dilpreettoor/dilpreet-toor-brainflix.git
```

Go to the project directory

```bash
  cd dilpreet-toor-brainflix
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
